## Bug #2: Windows 路径反斜杠导致转义错误

**版本**: v1.1.0  
**状态**: ✅ 已修复 (v1.1.1)

**修复方案**:
1. 在 `_extract_string_arg` 添加 `normalize_path` 参数
2. 路径参数自动将反斜杠 `\` 替换为正斜杠 `/`
3. 文件内容不进行路径规范化，保持原样
4. `_unescape` 方法添加错误处理，防止转义异常
